# 拼车社区

通过拼车方式分摊 API 使用成本，享受更高性价比的 AI 体验。

## 方案介绍

我们的拼车方案经过长期实践验证，提供原生级别的 API 使用体验：

- **专号专用** — 每个拼车组使用独立账号，互不干扰
- **用量透明** — 可随时查看自己的 Token 消耗情况
- **稳定可靠** — 高速专线服务器，分布式部署保障稳定性
- **合法合规** — 规范使用，专注于 AI 学习和开发

## 使用规则

为了保障所有成员的使用体验，请遵守以下规则：

1. **合理使用** — 专注于 AI 学习和开发，不得用于违规用途
2. **资源共享** — 避免长时间大量占用资源，尊重其他成员的使用权益
3. **保密 Key** — 不要将 API Key 分享给他人使用
4. **及时反馈** — 遇到问题及时联系管理员，不要自行排查敏感配置
